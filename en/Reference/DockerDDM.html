DockerHDDM 命令生成器

🐳 DockerHDDM 命令生成器

输入你的数据文件夹路径，一键生成可运行的 Docker 命令

📦 首次使用：拉取镜像

如果你的电脑还没有安装 DockerHDDM，请先在 CMD（管理员模式） 中运行： 

docker pull hcp4715/hddm
📁 输入你的数据文件夹路径
→
🚀 复制以下命令到 CMD（管理员模式）运行

      

      

        
docker run
启动一个 Docker 容器

        
-it
保持交互模式，可以看到 Jupyter 的输出日志

        
--rm
容器退出后自动删除，不占用磁盘空间

        
--cpus=N
限制容器使用的 CPU 核心数

        
-v 本地路径:容器路径
将你的数据文件夹挂载到容器内

        
-p 8888:8888
将容器的 8888 端口映射到本机

        
hcp4715/hddm:tag
使用的 Docker 镜像及版本

        
jupyter notebook
启动 Jupyter Notebook

      

      

        💡 使用提示：
        
    
          
  • 运行后，复制终端中显示的 http://127.0.0.1:8888/?token=... 到浏览器打开
    • 
          
  • 在 Jupyter 中找到 work 文件夹，里面就是你挂载的数据
    • 
          
  • 如果端口被占用，可以修改为 8787、8889 等其他端口
    • 
          
  • 每次用完关闭终端窗口即可，容器会自动清理（--rm）
    •
